# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly by emailing the maintainers rather than opening a public issue.

## Supported Versions

Security fixes are applied to the latest release on each supported OLM channel.

| Version | Supported |
|---------|-----------|
| v2.2.x  | Yes       |
| v2.1.x  | Yes       |
| v2.0.x  | Yes       |
